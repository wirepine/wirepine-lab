API Explorer

8 live public APIs — no sign-up, no keys. Hit Live to see fresh data, Code to see how it's called.

wheretheiss.at

🛸 ISS Tracker

Locating station… 
fetch('https://api.wheretheiss.at
  /v1/satellites/25544')
  .then(r => r.json())
  .then(d => {
    d.latitude   // e.g. 45.32
    d.longitude  // e.g. -87.21
    d.altitude   // km above Earth
    d.velocity   // speed in km/h
  });
api.nasa.gov

🔭 Photo of the Day

Contacting NASA… 
fetch('https://api.nasa.gov/planetary
  /apod?api_key=DEMO_KEY')
  .then(r => r.json())
  .then(d => {
    d.title        // photo title
    d.url          // image URL
    d.date         // YYYY-MM-DD
    d.explanation  // description
    d.media_type   // "image" or "video"
  });
opentdb.com

🎯 Trivia Challenge

Loading question… 
fetch('https://opentdb.com/api.php
  ?amount=1&type=multiple')
  .then(r => r.json())
  .then(d => {
    const q = d.results[0];
    q.question          // the question
    q.correct_answer    // right answer
    q.incorrect_answers // array of 3
    q.category          // e.g. "Science"
    q.difficulty        // easy/medium/hard
  });
pokeapi.co

⚡ Random Pokémon

Searching Pokédex… 
// IDs 1–898 are valid
const id = Math.floor(Math.random() * 898) + 1;

fetch(`https://pokeapi.co/api/v2/pokemon/${id}`)
  .then(r => r.json())
  .then(d => {
    d.name                   // "pikachu"
    d.sprites.front_default  // sprite URL
    d.types[0].type.name     // "electric"
    d.stats[0].base_stat     // HP value
  });
open-meteo.com

🌤 SF Weather

Checking skies… 
// No API key needed!
fetch('https://api.open-meteo.com/v1/forecast
  ?latitude=37.77&longitude=-122.42
  ¤t_weather=true
  &temperature_unit=fahrenheit')
  .then(r => r.json())
  .then(d => {
    const w = d.current_weather;
    w.temperature  // °F
    w.windspeed    // mph
    w.weathercode  // WMO weather code
  });
cataas.com

🐱 Cat GIF

Summoning a cat… 
// The URL *is* the response — no JSON!
const img = document.createElement('img');
img.src = 'https://cataas.com/cat/gif';
document.body.appendChild(img);

// Add ?t=Date.now() to bust the cache
// and get a fresh cat every time.
// Try: /cat/gif?type=square
// or:  /cat (returns a still photo)
dog.ceo

🐕 Random Dog

Fetching a good boy… 
fetch('https://dog.ceo/api/breeds/image/random')
  .then(r => r.json())
  .then(d => {
    d.message  // image URL
    d.status   // "success"
  });

// Breed is encoded in the URL path:
// .../breeds/[breed]/[file].jpg
// Try: /api/breed/husky/images/random
en.wikipedia.org

📖 Wikipedia Random

Opening a random article… 
fetch('https://en.wikipedia.org
  /api/rest_v1/page/random/summary')
  .then(r => r.json())
  .then(d => {
    d.title        // article title
    d.extract      // short summary text
    d.thumbnail    // {source, width, height}
    d.content_urls // .desktop.page = full URL
  });
