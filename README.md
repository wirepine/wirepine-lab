# Wirepine Lab

A static one-page website for an AI-native experimental studio.

## Structure

```
wirepine-lab/
├── index.html    # Semantic HTML structure
├── styles.css    # Modern CSS (Grid, Flexbox, custom properties)
└── README.md
```

## Sections

- **Hero** — Title and tagline introducing the studio
- **Manifesto** — Editorial text on the studio's philosophy (experiments, systems, AI, design)
- **Experiments** — Responsive grid of project cards
- **Featured Artifact** — Image placeholder with descriptive caption
- **Footer** — Links to wirepine.com and newsletter

## Technical Notes

- No JavaScript, frameworks, or build tools
- CSS custom properties for theming
- CSS Grid for experiments layout; Flexbox for alignment
- Responsive breakpoints at 480px, 768px, and 1024px
- Semantic HTML5 elements (`<header>`, `<main>`, `<section>`, `<article>`, `<figure>`, `<footer>`)

## Usage

Open `index.html` in any browser.
